Welcome to EventBook the Ultimate Scheduler Brought to you by MistyPanda

Even in our small community we are flooded with emails, posters and other ads for various events on campus. Our lives would be much easier and better served if there were some system that brings all of these events together. This goal can become a reality using the new Ultimate Scheduler, EventBook. This mobile app will be capable of consolidating all events on campus (club activities, sports, FAEâ€™s, etc.) in one common place. Not only will you be able to see an event, but also you will be able to upload pics taken on you device. This way you can be sure an event you are interested in is worth the hype.

EventBook capabilities

  1. View Events:
    The scheduler will allow you to view all of the future events stored in our database
  2. Search Events:
    If you're looking for a specific event you will be able to search for any event in our database
  3. Add Events:
    If there isnt an event you can make your own and send it to our database for your friends to see
  4. Edit Your Events:
    If you messed up entering it or if something changes you will also be able to edit your event after providing a password you choose during creation
  5. Remove Events:
    In a similar way you will also able to remove events you've created from the database
  6. Upload Pictures:
    In order to show how cool an event is you have the capabilities to upload pictures for your events as well
  7. Remove Events:
    In a similar way you will also able to remove events you've created from the database
